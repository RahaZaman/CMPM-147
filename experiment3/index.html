Experiment 3 - Alternate Worlds

Description

This page contains two projects. One project showcases an Overworld Map, dynamically generating biomes like grass and trees using Perlin noise, with moving clouds adding to its lifelike charm. The other project features a Dungeon Room generator, utilizing autotiling to construct randomized room layouts, delineating borders and interior tiles for an immersive dungeon experience. Both projects demonstrate the creative application of grid-based generation techniques in interactive environments.

Technical

The Dungeon Room and Overworld Map projects both leverage autotiling logic and grid-based generation to create immersive environments. In the Dungeon Room project, a randomized dungeon room is generated within the grid, with borders marked using autotiling. Each cell within the room is meticulously filled, utilizing a combination of symbols to distinguish between borders and room tiles. On the other hand, the Overworld Map project utilizes Perlin noise to generate biomes, with autotiling seamlessly transitioning between different terrain types. Additionally, dynamic elements such as moving clouds add a sense of life and dynamism to the overworld landscape. Overall, both projects showcase the versatility of grid-based generation and autotiling in creating interactive and visually appealing environments.
