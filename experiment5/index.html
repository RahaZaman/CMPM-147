Description

A parameterized image generator that adapts its designs based on three selectable inspiring images allows users to fine-tune exploration with a mutation rate slider.

Technical

  1. This project leverages the p5.js library to dynamically render canvas and manipulate images, ensuring a fluid creative process.
  2. JavaScript is employed to seamlessly integrate user-selected images, offering a diverse range of inspirations for design generation.
  3. An advanced random mutation algorithm is implemented to refine design exploration, enabling incremental changes and diverse outcomes.
  4. The design is rendered in a way that can possibly produce results that feel related to the inspiring images. (The 30-second annealing trick should produce images that resemble the inspiring image if we squint.)

Annealing

For best results, adjust the mutation rate gradually from 100% to 1% over about 30 seconds. This allows the generator to make significant changes initially and focus on finer details later. Monitor the outputs and use your judgment to determine when to make bigger or smaller changes manually. Finally, right-click on the best image to save it as a PNG file for future use.
